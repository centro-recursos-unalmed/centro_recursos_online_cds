README

The following are the recommended minimum system requirements:

* Pentium 266 MHz or faster processor
* Microsoft Windows 98, Me, 2000, or XP
* 128 MB RAM 
* 8x CD-ROM drive
* 16-bit video card with 800 x 600 resolution
* Sound card
* Mouse, speakers, microphone
* Printer (optional)
* Microsoft Windows Media Player

To get started, place the CD-ROM in your CD-ROM drive. On most computer systems, the program will start by itself. If the program does not start by itself, double-click on My Computer on your desktop or in the Start menu. Then double-click on the program icon.

For frequently asked questions and other information about the CD-ROM, go to www.cambridge.org/esl/cdrom.
